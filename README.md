# Data Science Service

A Python service that extracts structured data from emails using LangChain and Mistral AI.

## Features
- Connects to Gmail to fetch and parse emails
- Extracts structured information using LangChain + Mistral AI
- REST API endpoints via FastAPI

## Tech Stack
Python, LangChain, Mistral AI, FastAPI, Gmail API

## Setup
1. Clone the repo
2. Add `.env` with your API keys
3. Run: `pip install -r requirements.txt`
4. Run: `python setup.py`
